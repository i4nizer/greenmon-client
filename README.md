# Greenmon Client

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/i4nizer/greemon-clien.git
   ```
2. Navigate to the project directory:
   ```sh
   cd greenmon-client
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the server:
   ```sh
   npm start
   ```
